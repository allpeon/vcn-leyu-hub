# vcn-leyu-hub

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单个域名或具体网站，仅作为 HTML 静态页面的集中存放与展示空间。

## 项目简介

vcn-leyu-hub 旨在收集、整理并托管一系列独立的 HTML 页面。这些页面可能涵盖各类主题或功能，彼此之间无强关联，以文件形式独立存在。仓库本身不附带特定业务逻辑或后端服务，所有内容均为纯前端静态页面。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（如 CSS、JS、图片等）
├── docs/           # 相关文档或说明文件（可选）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件代表一个独立页面，可直接通过浏览器或 GitHub Pages 访问。
- `assets/`：按需存放页面引用的样式表、脚本、字体、图片等资源。
- `docs/`：存放与页面使用、部署或背景相关的补充文档。

## 页面归档说明

本仓库中的 HTML 页面以文件为单位归档，每个页面通常包含完整的 HTML 结构及内联或引用的样式与脚本。页面之间互不依赖，可单独打开使用。

归档的页面可能来自不同时期、不同来源，或为自行编写。仓库仅对页面进行格式整理与托管，不保证所有页面的持续更新或兼容性。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中直接打开 `pages/` 目录下的任意 HTML 文件。
3. 若仓库启用了 GitHub Pages，可通过对应 URL 访问 `pages/` 下的页面。
